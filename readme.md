

# Monochroma
Dark-Mode made Simple in React Apps.

## Technologies Used - 
- **ReactJS**
- **NodeJs**

## Installation
```
yarn add monochroma
```

## Usage
- Import Monochroma
```
 import Monochroma from 'monochroma';
```
- Wrap Your Component inside Monochroma
```
<Monochroma>
    <App />
</Monochroma>
```
